## Installation

1. **Clone the Repository**
    ```bash
    git clone https://github.com/yourusername/cuda-convolution.git
    cd cuda-convolution
    ```

2. **Create a Build Directory**
    ```bash
    mkdir build
    cd build
    ```

3. **Run CMake**
    ```bash
    cmake ..
    ```

4. **Build the Project**
    ```bash
    make
    ```

5. **Run Tests**
    ```bash
    ctest
    ```
